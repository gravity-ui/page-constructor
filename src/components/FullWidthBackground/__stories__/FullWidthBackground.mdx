## Usage

For a detailed usage guide of the FullWidthBackground component, see [FullWidthBackground Usage](https://github.com/gravity-ui/page-constructor/blob/main/memory-bank/usage/fullWidthBackground.md).

## Parameters

`className?: string` — Optional CSS class name for additional styling

`style?: React.CSSProperties` — Optional inline styles

`theme?: 'default' | 'rounded'` — Theme variant ('default' by default)
- `'default'`: Creates a full-width background without border radius
- `'rounded'`: Creates a full-width background with rounded corners and responsive padding

`children?: React.ReactNode` — Optional child elements to render inside the background
