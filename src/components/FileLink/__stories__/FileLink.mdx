## Usage

For a detailed usage guide of the FileLink component, see [FileLink Usage](https://github.com/gravity-ui/page-constructor/blob/main/memory-bank/usage/fileLink.md).

## Parameters

`href: string` — File URL or path (required)

`text: React.ReactNode` — Link text content (required)

`type?: 'vertical' | 'horizontal'` — Layout type ('vertical' by default)

`textSize?: TextSize` — Text size - 'xs', 's', 'sm', 'm' (default), or 'l'

`theme?: ContentTheme` — Content theme - 'default', 'dark', or 'light'

`className?: string` — Optional CSS class name

`urlTitle?: string` — Optional title attribute for the link

`onClick?: () => void` — Optional click handler function

`tabIndex?: number` — Optional tab index for accessibility

`extraProps?: React.HTMLProps<HTMLAnchorElement>` — Optional additional HTML anchor props
