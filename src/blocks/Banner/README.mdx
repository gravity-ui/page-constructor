## Banner block params

`type: 'banner-block'`

`animated?: bool` — Enables/disables animation for the block (enabled by default).

`title: string` — Title

`subtitle?: string` — Subtitle

`image?: url` — Image on the right

`color?: string` — Background color

`button: Button` — Button
