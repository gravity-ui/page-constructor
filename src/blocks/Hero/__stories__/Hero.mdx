## Parameters

There could be only one Hero block on the page.

`type: 'hero-block'`

`breadcrubms?: HeaderBreadCrumbsProps` — Breadcrumbs props (refer to `HeaderBreadcrumbs` component).

`overtitle?: string | JSX.Element` — Overtitle text that is shown above the title.

`title: string` — Title.

`text?: string` — Subtitle text.

`buttons?: ThemeSupporting<Pick<ButtonProps, 'url' | 'text' | 'theme' | 'primary' | 'extraProps'>>[]` — List of buttons (refer to `Button` component).

`media?: MediaProps & {roundCorners?: boolean}` — Displayed media properties (refer to `Media` component).

`fullWidth?: boolean` — Whether block background takes up all the screen width or not (default `false`)

`verticalOffset?: 's' | 'm' | 'l' | 'xl` — The size of top and bottom paddings around the block's text content.

`theme?: 'light' | 'dark'` — Color theme.

`background?: ThemeSupporting<HeroBlockBackground>` — Block background properties (refer to `Media` component; only color, image and video properties apply)

</StoryTemplate>
